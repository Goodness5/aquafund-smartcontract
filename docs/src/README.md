# AquaFund Smart Contract Documentation

Welcome to the AquaFund Smart Contract documentation! This guide is designed for **frontend developers** who want to integrate with the AquaFund platform.

## What is AquaFund?

AquaFund is a decentralized crowdfunding platform built on Ethereum that enables transparent, secure, and efficient funding for water-related projects. The platform uses smart contracts to manage donations, track project progress, and reward donors with NFT badges.
